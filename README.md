# VOS

VOS is a lightweight, modular operating-system-inspired project scaffold.

## Structure

- apps/: built-in applications such as browser, calculator, explorer, notes, settings, and terminal.
- core/: shared infrastructure including logging, events, configuration, and services.
- runtime/: lifecycle management for applications, processes, and sessions.
- desktop/: desktop shell components such as taskbar, start menu, and window manager.
- docs/: architecture notes, ADRs, roadmap, changelog, and coding standards.

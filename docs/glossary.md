# VOS Glossary

## Core Terms

- VOS: A modular, operating-system-inspired project scaffold.
- Kernel: The central control component that coordinates system operations.
- Service Manager: A registry for shared system services such as logging and configuration.
- Event Bus: A communication mechanism for decoupled event-based interaction between components.
- Runtime: The layer responsible for application lifecycle, process tracking, and session management.
- Desktop Shell: The user-facing environment that hosts the desktop, taskbar, and menus.

## Project-Specific Terms

- App Manager: A component that registers and resolves built-in applications.
- Process Table: A structure for tracking running processes by identifier.
- Session Manager: A component that manages active user sessions.
- Boot Sequence: The ordered initialization flow used to start core subsystems.
